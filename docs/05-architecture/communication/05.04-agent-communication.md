# 05.04 Agent Communication

## Overview

The Agent Communication Architecture defines how AI Agents inside cmdOS exchange information, coordinate tasks, share context, and collaborate during execution.

Because cmdOS is designed as an AI-native operating system, multiple Agents may operate together to complete complex objectives.

Agent Communication provides the coordination layer between autonomous intelligence units.
