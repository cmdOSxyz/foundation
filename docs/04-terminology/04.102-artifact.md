# 1. Purpose

This document defines the canonical meaning of **Artifact** within cmdOS.

Artifact is the canonical representation of any tangible output, generated object, modified resource, or created asset produced through Actions, Executions, Workflows, Agents, or Systems.

Artifacts represent the physical results created by intelligence operating through cmdOS.

Every meaningful creation process within cmdOS shall produce one or more Artifacts.

---

# 2. Definition

An **Artifact** is a persistent object created, modified, or managed as the result of an operation performed within cmdOS.

An Artifact defines:

- created object
- source operation
- ownership
- location
- version
- metadata
- verification status

Artifact represents something produced or changed.

---
