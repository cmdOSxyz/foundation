# Context

Version: 1.0  
Status: Canonical  
Category: Foundation

---

# 1. Purpose

This document defines the canonical meaning of **Context** within cmdOS.

Context is the canonical information environment that provides surrounding conditions, historical information, current state, and relevant circumstances required for understanding Intent, making Decisions, and performing Actions.

Context enables cmdOS to interpret information correctly based on situation rather than isolated input.

Every intelligent operation within cmdOS shall operate with appropriate Context.

---

# 2. Definition

**Context** is the collection of relevant information describing the conditions surrounding an Entity, Event, Intent, Decision, or Execution.

Context defines:

- current situation
- historical information
- environmental conditions
- relationships
- constraints
- available resources
- user state

Context provides meaning to information.

---

# 3. Context Is Not Data

Data represents raw information.

Context represents information with relevance and meaning.

```
Data

↓

Context

↓

Understanding
```

Context explains why information matters.

---

# 4. Context Is Not Memory

Memory stores retained information.

Context selects relevant information for the current situation.

```
Memory

↓

Context

↓

Reasoning
```

Memory provides availability.

Context provides relevance.

---

# 5. Context Is Not State

State represents the current condition of a system.

Context may include State plus additional information.

```
State

+

Environment

+

History

↓

Context
```

---

# 30. Summary

Context provides the situational awareness layer of cmdOS.

It connects Intent, Memory, Knowledge, Reasoning, Decision, Planning, and Execution while allowing AI systems to understand situations rather than process isolated commands.

By separating Context from Data, Memory, and State, cmdOS establishes the foundation for intelligent systems capable of understanding users, environments, and changing conditions.

Context is therefore the canonical situational understanding primitive of the AI Execution Operating System.
