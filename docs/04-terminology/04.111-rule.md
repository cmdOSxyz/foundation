# 1. Purpose

This document defines the canonical meaning of **Rule** within cmdOS.

Rule is the smallest executable logic unit within the Governance framework responsible for evaluating conditions, determining outcomes, and enforcing operational behavior.

Rules transform Policies into concrete decisions.

Every Policy within cmdOS shall be composed of one or more Rules.

---

# 2. Definition

A **Rule** is a formal logical statement that defines a condition and the expected behavior when that condition is satisfied.

Rule defines:

- condition
- evaluation logic
- expected outcome
- enforcement action
- priority
- scope

Rule determines what happens under specific circumstances.

---
