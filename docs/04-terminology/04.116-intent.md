# Intent

Version: 1.0  
Status: Canonical  
Category: Foundation

---

# 1. Purpose

This document defines the canonical meaning of **Intent** within cmdOS.

Intent is the canonical representation mechanism responsible for capturing the desired purpose, request, motivation, or objective expression from a User, Agent, or System before conversion into structured Goals, Plans, Decisions, and Executions.

Intent is the primary human-to-machine interaction layer of the AI Execution Operating System.

Every meaningful operation within cmdOS begins with understanding Intent.

---

# 2. Definition

An **Intent** is the underlying purpose, desire, or expected outcome expressed by an Entity that initiates a process within cmdOS.

Intent defines:

- desired outcome
- motivation
- expectation
- priority
- context
- constraints

Intent represents what someone wants to achieve.

---

# 3. Intent Is Not Command

A Command specifies an exact operation.

Intent describes the desired result without requiring specific instructions.

Example:

Command:

```
Open Chrome
```

Intent:

```
I need to research competitors
```

cmdOS converts Intent into required Actions.

---

# 4. Intent Is Not Goal

Intent represents the initial desire.

Goal represents the structured and measurable outcome derived from that desire.

```
Intent

↓

Goal

↓

Planning
```

Intent is human expression.

Goal is operational objective.

---

# 5. Intent Is Not Action

Intent explains why.

Action defines what operation will happen.

```
Intent

↓

Decision

↓

Action

↓

Execution
```

---

# 32. Summary

Intent provides the human-to-AI interaction layer of cmdOS.

It connects Users, Goals, Planning, Decisions, Actions, Execution, Results, and Learning while allowing humans to communicate desired outcomes without requiring technical instructions.

By separating Intent from Command, Goal, and Action, cmdOS establishes a foundation for a true AI Operating System where users express what they want and the system determines how to achieve it.

Intent is therefore the canonical human interaction primitive of the AI Execution Operating System.
