# Skill Graph

Version: 1.0  
Status: Canonical  
Category: Foundation

---

# 1. Purpose

This document defines the canonical meaning of **Skill Graph** within cmdOS.

Skill Graph is the canonical relationship structure responsible for representing, organizing, discovering, and connecting Skills, Capabilities, Tools, Agents, and Execution patterns.

Skill Graph enables cmdOS to dynamically understand available abilities and determine optimal combinations required to achieve Goals.

Every advanced Agent capability within cmdOS shall rely on Skill Graph representation.

---

# 2. Definition

A **Skill Graph** is a dynamic network structure that represents relationships between Skills, Capabilities, Tools, Agents, Resources, and Execution Methods.

Skill Graph defines:

- available abilities
- relationships between abilities
- dependencies
- execution paths
- compatibility
- performance history

Skill Graph represents how abilities connect.

---

# 3. Skill Graph Is Not Capability

Capability represents an ability category.

Skill Graph represents the network connecting multiple abilities.

```
Capability

↓

Skill Graph

↓

Execution Path
```

---

# 4. Skill Graph Is Not Skill

Skill represents a specific ability or method.

Skill Graph represents relationships between many Skills.

```
Skill

+

Skill

+

Skill

↓

Skill Graph
```

---

# 5. Skill Graph Is Not Workflow

Workflow represents a predefined execution process.

Skill Graph dynamically discovers possible execution paths.

```
Workflow

↓

Known Process
```

```
Skill Graph

↓

Adaptive Process Discovery
```

---

# 6. Skill Graph Characteristics

Every Skill Graph shall be:

- dynamic
- extensible

---

# 33. Summary

Skill Graph provides the ability intelligence layer of cmdOS.

It connects Capabilities, Skills, Tools, Agents, Planning, Execution, Memory, and Learning while allowing autonomous systems to discover and improve their own capabilities.

By separating Skill Graph from Skill, Capability, and Workflow, cmdOS establishes the foundation for a self-expanding AI Operating System ecosystem.

Skill Graph is therefore the canonical ability relationship primitive of the AI Execution Operating System.
